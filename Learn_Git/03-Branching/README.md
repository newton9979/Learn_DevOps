# Learning Flow

```text
Create Branch
      ↓
Switch Branch
      ↓
Checkout
      ↓
Merge
      ↓
Rebase
      ↓
Cherry-Pick
      ↓
Delete Branch
      ↓
Merge Conflict
```

This roadmap summarizes the complete Git Branching workflow used by DevOps Engineers, Cloud Engineers, SREs, and Software Developers in real-world projects.
