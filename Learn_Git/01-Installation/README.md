# 📦 01 - Installing Git

Welcome to **Day 2** of the **Learn Git** series.

This section covers everything you need to install Git on different operating systems, verify the installation, configure Git for first-time use, and understand essential Git configuration commands.

---

## 🎯 Learning Objectives

After completing this module, you will be able to:

* Install Git on Windows
* Install Git on Linux
* Install Git on macOS
* Verify the Git installation
* Configure Git for first-time use
* Understand Git configuration levels
* Use Git help commands
* Troubleshoot common installation issues
